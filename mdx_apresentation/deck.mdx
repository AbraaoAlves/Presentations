o que é markdown

Markdown é uma linguagem de marcação de texto simples que permite formatar e estilizar documentos de forma rápida e fácil. Com o Markdown, é possível adicionar negrito, itálico, listas, links, imagens e outros elementos em um texto, sem precisar digitar HTML. O Markdown é amplamente utilizado em plataformas de blogs, wikis, fóruns e outras formas de conteúdo web para criar documentos bem formatados e fáceis de ler.

---

quem criou?
O Markdown foi inventado em 2004 por John Gruber, um escritor e desenvolvedor web americano, em colaboração com Aaron Swartz, um programador e ativista da internet. Gruber criou o Markdown como uma forma mais simples e fácil de criar documentos bem formatados para a web, sem precisar escrever HTML ou outros códigos mais complexos. Desde então, o Markdown se tornou uma das linguagens de marcação mais populares na web, sendo utilizada por muitos programadores, escritores e profissionais de comunicação.

o que é o mdx?

MDX é uma extensão da linguagem Markdown que permite incluir componentes React em seus arquivos Markdown. Com o MDX, é possível criar documentos mais interativos, com funcionalidades avançadas, como gráficos, tabelas dinâmicas e outras visualizações de dados. O MDX é amplamente utilizado para criar documentação, apresentações, blogs e outras formas de conteúdo web.

---

quem inventou?
O MDX foi criado em 2017 por Christopher "vjeux" Chedeau, um engenheiro de software do Facebook, como uma extensão do Markdown que permite incluir componentes React em arquivos Markdown. O objetivo do MDX é permitir a criação de conteúdo web mais interativo e funcional, que pode incluir gráficos, tabelas dinâmicas e outras visualizações de dados. Desde então, o MDX se tornou uma ferramenta popular entre desenvolvedores web e escritores que desejam criar conteúdo mais avançado e dinâmico.

---

quem apoia?
O MDX é apoiado por uma ampla comunidade de desenvolvedores, escritores e empresas que reconhecem os benefícios de combinar as funcionalidades do React com a simplicidade do Markdown. Alguns dos principais defensores do MDX incluem empresas como a Gatsby, que utiliza o MDX para criar sites estáticos avançados, e a Amazon Web Services (AWS), que utiliza o MDX para criar documentação interativa e avançada. Além disso, o MDX é amplamente utilizado em projetos open source, incluindo o próprio projeto React da Facebook, bem como em outras bibliotecas e frameworks do ecossistema React.
